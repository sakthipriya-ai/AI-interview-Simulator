# AI Interview Simulator & Resume Analyzer

An AI-powered web platform that helps users practice technical interviews and analyze resumes using machine learning.

---

## Features

- AI Interview Question Simulator
- Resume Analyzer with Skill Detection
- Voice Based Answer System
- Performance Analytics Dashboard
- Resume Scoring using NLP
- User Authentication System

## Installation

Clone the repository

git clone https://github.com/yourusername/ai-interview-platform

Install dependencies

pip install flask pdfplumber python-docx sentence-transformers scikit-learn

Run the application

python app.py
